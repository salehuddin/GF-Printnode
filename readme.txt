Self-contained plugin that generates PDF labels from GF submissions and sends them to PrintNode. Includes a tracking dashboard.

== Description ==

Gravity Forms Smart Print & Tracker integrates seamlessly with Gravity Forms to automatically generate high-fidelity PDF thermal labels upon form submission. It routes these labels directly to your physical printers via the PrintNode API, completely bypassing the browser's print dialog.

Includes a comprehensive "Print Logs" dashboard to track every print job, view generated PDFs, and reprint failed jobs with a single click.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/gravity-forms-printnode` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Configure your PrintNode API Key in the Gravity Forms Add-On Settings page.
4. Set up a PrintNode Feed in your desired Gravity Form.

== Frequently Asked Questions ==

= Do I need a PrintNode account? =
Yes, you will need an active PrintNode account and an API key to communicate with your registered printers.
